## Overview 🌟
Welcome to the **Finance Dashboard**! 🎉 This Power BI dashboard 🚀 visualizes financial data from 2021 to 2024 🗓️. It features **dynamic slicers** 🎚️ for monthly growth 📈 and KPIs, a visually stunning **pie chart** 🥧 with interactive tooltips 😎 for expenses 💸 and savings 🤑, and **dynamic titles** 📝 driven by DAX measures. Many KPIs are crafted using DAX for enhanced flexibility and insights.

## Features ✨
- **Dynamic Slicers** 🎛️: Filter by monthly growth 📈, years (2021–2024) 🗓️, or other KPIs 🔥.
- **Interactive Pie Chart** 🥧: Shows expenses 💵 and savings 🏦 with vibrant tooltips 😍.
- **Dynamic Titles** 📝: Automatically update based on slicer selections using DAX measures.
- **DAX-Driven KPIs** 📊: Custom metrics (e.g., growth rates, expense ratios) calculated via DAX.
- **Time Range** ⏳: Covers 2021 to 2024 🕰️ for trend analysis.
- **Expenses by Date** 📅: Visualize spending trends 📈 over time.

## Prerequisites 🛠️
- **Power BI Desktop** or **Power BI Service** 🖥️ (latest version recommended).
- Financial dataset 📈 in CSV, Excel, or database format with columns for:
  - Date 📆 (e.g., MM/DD/YYYY)
  - Expenses 💰 (numeric values)
  - Savings 🤑 (numeric values)
  - Other KPIs 📊 (previously revenue, profit).
- Basic knowledge of Power BI and DAX for customization 🔧.

## Installation Steps 🛠️
1. Open the `Finance_Dashboard.pbix` file in Power BI Desktop 🖱️.
2. Connect to your financial dataset 📊:
   - Go to **Home** > **Get Data**.
   - Import CSV, Excel, or database file 🔗.
   - Map fields to dashboard visuals 📈.
3. Save and publish to Power BI Service (optional) ☁️.

## Usage 🎉
1. Open the dashboard in Power BI Desktop or Service 🚀.
2. Use **slicers** 🎚️ to filter by year (2021–2024) 🗓️ or growth 📈.
3. Hover over the **pie chart** 🥧 for expense 💸 and savings 🤑 tooltips 😍.
4. Check **Expenses by Date** tab 📅 for spending trends 📈.
5. Observe **dynamic titles** 📝 updating based on slicer selections.
6. Explore DAX-driven KPIs 📊 for deeper insights.

## Customization 🖌️
- **Data Source** 🔗: Update dataset in Power BI’s **Data View**.
- **Slicers** 🎛️: Add slicers for new KPIs via **Fields** pane.
- **DAX Measures** 📝: Modify or create new DAX measures in **Model View** for custom KPIs or titles.
- **Visuals** 🖼️: Adjust pie chart 🥧 colors, fonts, or add new visuals (e.g., bar 📊 or line graphs 📈).
- **Tooltips** 😎: Enhance tooltip content with additional DAX metrics.
- **Pages** 📑: Add tabs for specific analyses (e.g., category-wise expenses).
